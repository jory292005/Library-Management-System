# 📚 Library Management System

A Java-based console application for managing library operations, including book borrowing, returning, reservation, and user administration.

## 🚀 Key Features
* **User & Admin Roles:** Different privilege levels for managing the system.
* **Book Management:** Admin can add new books to the library.
* **Borrowing & Returning:** Users can borrow available books and return them.
* **Reservation System:** Users can reserve books that are not currently available.
* **Transaction Tracking:** Logs borrowing/returning activities.

## 🛠️ Technical Details
* **Language:** Java
* **Programming Paradigm:** Object-Oriented Programming (OOP)
* **Data Structures:** Used `ArrayList` to manage books, users, and transactions.
